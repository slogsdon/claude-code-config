Generate production-ready frontend templates for landing pages, dashboards, and components with modern design standards (2024-2025), WCAG 2.2 AA accessibility compliance, and performance optimization.

## Requirements

**Technical Standards**:
- Semantic HTML landmarks (header, nav, main, aside, footer)
- WCAG 2.2 AA compliance (color contrast ≥ 4.5:1, keyboard navigation, ARIA attributes)
- Responsive breakpoints (360px, 768px, 1024px, 1280px)
- Performance optimization (lazy loading, critical CSS, responsive images)
- SEO meta tags and structured data
- Dark mode support via prefers-color-scheme
- Respect prefers-reduced-motion for animations

**Design Standards**:
- Modern 2024-2025 aesthetics with clean layouts, soft shadows, rounded corners
- Clear information architecture with prominent CTAs
- Appropriate placeholder content matching industry tone
- Typography hierarchy with system fonts or specified web fonts
- Brand color integration

**Accessibility**:
- Skip links for keyboard users
- Proper heading hierarchy (h1-h6)
- Focus management and visible focus states
- Screen reader support with ARIA labels
- Form labeling and error association
- Meaningful image alt text

**Output Format**:
Structure responses as individual code files using:
```txt
# FILE: path/to/file.ext
[file contents]
```

Include a minimal README.md with setup and customization instructions.

**Before generating**, clarify: OUTPUT_SCOPE (landing_page/dashboard/component), PROJECT_STACK (react-jsx/laravel-blade-livewire), CSS_SYSTEM (tailwind/vanilla), INDUSTRY_TONE, BRAND colors/fonts, and specific requirements.
