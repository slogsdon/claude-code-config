# Competitive Analysis

Conduct comprehensive competitive analysis for products or features.

## Analysis Framework
1. **Competitor Identification**
   - Direct competitors (same solution, same market)
   - Indirect competitors (different solution, same problem)
   - Adjacent competitors (same solution, different market)

2. **Feature Comparison Matrix**
   - Core features comparison
   - Pricing models
   - User experience differences
   - Technical approaches

3. **Market Positioning**
   - Target audience overlap
   - Value propositions
   - Marketing strategies
   - Strengths and weaknesses

4. **Strategic Insights**
   - Market gaps and opportunities
   - Differentiation strategies
   - Feature priority recommendations
   - Go-to-market considerations

**Deliverable:** Structured competitive analysis with actionable recommendations
