# Define New Product/Feature

Structure the definition of a new product or feature using product management best practices.

## Definition Framework
1. **Problem Statement**
   - What problem are we solving?
   - Who has this problem?
   - How big is the problem? (market size, frequency)

2. **Solution Hypothesis**
   - Proposed solution approach
   - Key assumptions to validate
   - Success metrics

3. **Product Requirements Document (PRD)**
   - Executive summary
   - Goals and success metrics
   - User personas and use cases
   - Feature specifications
   - Technical requirements
   - Go-to-market strategy

4. **Validation Plan**
   - User research methods
   - Prototype/MVP approach
   - Testing strategy
   - Launch plan

**Output:** Complete PRD with validation plan
