# Skill: /emerge [domain]

- Step 1: Search vault for all notes in [domain]. If no argument, full vault scan.
- Step 2: Identify scattered premises — things mentioned repeatedly but never connected
- Step 3: Find unnamed patterns — recurring behaviors, decisions, or themes without a label
- Step 4: Draw 3–5 conclusions the vault implies but has never stated. These should feel like "oh, I already believed this, I just never said it"
- Output: Emergent ideas with supporting vault evidence for each
