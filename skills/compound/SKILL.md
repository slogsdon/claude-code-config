# Skill: /compound [question]

- Step 1: Search vault, sorting notes chronologically. Divide into three periods: early (oldest third), middle, current (newest third).
- Step 2: Answer [question] using ONLY early vault context
- Step 3: Answer the same question using middle-period context
- Step 4: Answer using current context
- Step 5: Show what changed and why — what new knowledge or experience shifted the answer?
- Output: Three answers labeled by period, followed by a "what compounded" summary
