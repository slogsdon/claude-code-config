# Stakeholder Communication

Create effective communication materials for different stakeholder groups.

## Communication Strategy
1. **Audience Analysis**
   - Executive summary (high-level impact, ROI)
   - Engineering teams (technical requirements, timeline)
   - Design teams (user experience requirements)
   - Marketing/Sales (positioning, competitive advantage)

2. **Message Tailoring**
   - Key message for each audience
   - Supporting data and evidence
   - Call to action
   - Next steps

3. **Communication Formats**
   - Executive briefing (1-page summary)
   - Technical specification document
   - User story mapping
   - Roadmap presentation

4. **Follow-up Plan**
   - Feedback collection method
   - Decision timeline
   - Iteration process

**Output:** Tailored communication materials for specified stakeholders
