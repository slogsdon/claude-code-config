# Skill: /trace [topic]

- Step 1: Build a synonym vocabulary for [topic] — what other words or concepts would capture the same idea?
- Step 2: Search vault using all vocabulary terms. Look for dates/timestamps in note metadata to establish chronology.
- Step 3: Build the arc — earliest mention → middle period → current thinking. Note turning points and what caused them.
- Output: Chronological evolution with key turning points highlighted. Quote the vault directly at each inflection.
